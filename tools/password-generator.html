Password Security Tips

  • Use a unique password for every account — reusing passwords is the #1 cause of breaches
  • Store passwords in a password manager (Bitwarden, 1Password, or your browser's built-in manager)
  • Longer is always stronger — a 20-character password is exponentially harder to crack than a 10-character one
  • Enable two-factor authentication wherever possible for an extra layer of security
  • Never share passwords via email, SMS, or unencrypted apps
