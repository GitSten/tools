Vim Cheat Sheet

Core Vim movement, editing and save/quit commands for faster terminal editing.

ModesMovementEditingSave and quit

Modes

i insert mode
Esc normal mode
v visual mode
: command mode

Movement

h left
j down
k up
l right
w next word

Editing

dd delete line
yy copy line
p paste
u undo
Ctrl + r redo

Save and quit

:w save
:q quit
:wq save and quit
:q! quit without saving

Search and windows

/term search forward
?term search backward
n next match
N previous match
:split
:vsplit
Ctrl + w w switch window
